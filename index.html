What is Bit (B1T)? ‚ñº

To start mining B1T, you need SCRYPT-capable mining hardware. You can join a mining pool or mine solo if you have sufficient hashpower. Visit our GitHub page for guides on setting up a node and mining client.

What are the advantages of B1T compared to Bitcoin? ‚ñº
